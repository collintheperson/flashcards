Flash Cards

JavaScript(JS)

A scripting language to create interactive effects within a website.

Operator

A character that represents an action.

Variables

Symbolic name given that permits the name to be used independent of the information it represents.

Variable naming conventions

Using different names to create variables that can be used to create a function.

Functions

A named section of a program that performs a specific task.

Methods

A programmed procedure that is defined as part of a class and included in any object of that class.

Arguments

Independent items, or variables, that contain data or codes.

Parameters

A special kind of variable, used in a subroutines to refer to one piece of data.

Return

A statement that causes an execution to leave the current spot and resume at a point in the codes after where that spot is.

Chaining methods

Common syntax for invoking multiple method calls in object-oriented programming.

Strings

A string is traditionally a sequence of characters, usually some sort of variable.

Booleans

A binary variable, having two possible values "true" and "false".

Undefined

A condition where an expression does not have a correct value, although it is syntactically correct.

NaN

Stands for not a number and is a result of undefined operations on floating point numbers.

Escape

An escape character also forms some escape sequences which are referred to control characters.

Alert

A pop up screen to alert a user of a message that is important.

Comments

A readable explanation in the source code to explain what is happening.

jQuery

A concise and fast JavaScript library that can be used to simplify event handling.
